Directivas
    Las directivas son atributos que se pueden agregar a los elementos html, que nos sirven para aplicar transformaciones para los mismos elementos.

    v-show

    Mostrar/Ocultar párrafo

    {{ paragraph }}

    v-if

    Agregar/Eliminar párrafo

    {{ paragraph2 }}

    v-else

    {{ paragraph3 }}

    Nota: También podemos utlizar v-else-if y colocar otra condición para que se muestre este último párrafo.
