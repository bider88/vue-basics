Data Binding
Con el data binding nos permite enlazar datos de nuestra vista al código o viceversa. Quiere decir que cada cambio se verá reflejado de manera inmediata ya sea en código o la vista.

v-model

Es una directiva que nos permite enlazar a nuestor modelos los datos dentro de las etiquetas input, textarea y select.

model: {{ name }}

v-bind

Nos permite enlazar cualquier atributo HTML con cualquiera de las propiedades de nuestro v-model.

@didiersa

El anterior botón contiene un atributo que esta bindeado, es decir que el valor del href tiene enlazdo un modelo y lo se lo pasa al href por este medio v-bind:href. Una manera de dejarlo implicítamente es dejar solamente :href (shorthand).
